DeepSeek API

Get your API key at platform.deepseek.com → API Keys → Create new key.

The key powers all AI chat, task analysis, and recommendations.

Trello API

Get credentials at Trello Power-Ups Admin

  1. Create a Power-Up → get the API Key
  2. Click "Token" link to generate your token
  3. Paste both below and click Test Connection

Jira API

  1. Log in → Account Settings → Security → API Tokens
  2. Create token, copy it
  3. Your domain: yourcompany.atlassian.net

Slack API

  1. Go to api.slack.com/apps → Create App
  2. Add scopes: channels:read, channels:history, users:read
  3. Install to workspace → copy Bot User OAuth Token (xoxb-…)

Monday.com API

  1. Log in → Avatar → Admin → API
  2. Copy your personal API token
  3. Optional: get a Board ID from your board URL

Asana API

  1. Go to Asana Developer Console
  2. Create a Personal Access Token
  3. Find your Workspace GID in workspace settings
